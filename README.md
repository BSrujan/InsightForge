# InsightForge Analytics 📊

InsightForge Analytics is an interactive data visualization dashboard built using Python, Gradio, MySQL, Pandas, and Matplotlib.

The project dynamically fetches data from a MySQL database and generates multiple graph visualizations such as:
- Line Charts
- Bar Graphs
- Scatter Plots
- Pie Charts

## 🚀 Features

- Dynamic graph generation
- MySQL database integration
- Interactive Gradio UI
- CSV export support
- Row-wise data inspection
- Date-based filtering
- Real-time data loading

## 🛠️ Technologies Used

- Python
- Gradio
- Pandas
- NumPy
- Matplotlib
- MySQL Connector

## ▶️ Run Project

```bash
pip install -r requirements.txt
python final.py
```

## 📈 Graph Types Supported

- Line Graph
- Bar Chart
- Scatter Plot
- Pie Chart
